Customer Segmentation using Machine Learning

A machine learning web application built with *Python* and *Streamlit* that segments customers based on purchasing behavior using clustering techniques, helping businesses understand customer groups for targeted marketing.

## 🚀 Features

- Upload customer dataset
- Automatic data preprocessing
- Customer segmentation using K-Means Clustering
- Interactive data visualizations
- Real-time prediction through Streamlit

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## ⚙️ Installation

bash
git clone https://github.com/hima065/customer-segmentation.git
cd customer-segmentation
pip install -r requirements.txt
streamlit run app.py
